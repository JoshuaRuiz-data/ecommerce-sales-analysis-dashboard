# E-commerce Sales & Customer Analysis Dashboard

## Overview
This project analyzes e-commerce transaction data to uncover key business insights related to sales performance, product trends, and customer behavior.

## Tools Used
- Power BI
- Power Query
- DAX

## Key Features
- Revenue trend analysis
- Top-performing product analysis
- Customer segmentation
- Revenue contribution analysis
- Interactive filters and KPIs

- High-value customers drive a disproportionate amount of revenue
